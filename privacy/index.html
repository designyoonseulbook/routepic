개인정보 처리방침

최종 업데이트: 2026년 7월 9일

RoutePic은 오프라인 우선(Offline-First) 여행 기록 앱입니다.
앱은 자체 개발자 서버, 광고 SDK, 분석 SDK, 클라우드 동기화 기능을 사용하지 않습니다.
다만 지도 표시, 음성 인식, 번역 언어팩 다운로드, 사용자가 직접 실행한 메일·공유 동작에서는 외부 서비스 또는 외부 앱과 통신이 발생할 수 있습니다.

1. 앱이 접근하는 정보와 사용 목적

RoutePic은 위치, 카메라, 마이크, 사진 라이브러리, 음성 인식, 알림 등 필요한 시스템 권한을 사용할 수 있으며, 아래 정보를 처리합니다.

  • 위치 정보 — 여행 경로 기록, 장소 근접 알림, 사진·영상·음성 메모의 위치 메타데이터 저장에 사용됩니다. 진행 중인 여행에서는 백그라운드 위치 기록이 포함될 수 있습니다.
  • 사진·영상 및 관련 메타데이터 — 사용자가 촬영하거나 선택한 미디어와 그 촬영 시각, 위치 메타데이터 등이 포함될 수 있습니다.
  • 음성 데이터 및 음성 인식 결과 — 음성 메모 파일과 이를 텍스트로 변환한 결과가 포함될 수 있습니다.
  • 텍스트 추출 및 번역 결과 — 사진에서 추출한 텍스트와 번역 결과가 포함될 수 있습니다.
  • 사용자 입력 및 설정 정보 — 메모, 제목, 기본 내보내기 이메일 주소, 지도·번역·알림·단위 등 설정값이 포함될 수 있습니다.
  • 사용자 파일 — 사용자가 가져오거나 내보내는 KML, 오프라인 지도(.mbtiles), SQLite 백업 파일을 처리할 수 있습니다.
  • 진단 로그 — 앱 동작 과정에서 오류 확인과 상태 진단을 위한 로컬 로그가 생성될 수 있습니다.

2. 정보 저장 방식

  • 처리되는 정보는 기본적으로 앱 샌드박스 내부의 로컬 데이터베이스와 Documents 폴더에 저장됩니다. 여기에는 여행 기록, 사진·영상, 음성 메모, 음성 인식 결과, OCR/번역 결과, 가져온 파일, 로그, 앱 설정값이 포함될 수 있습니다.
  • 주 저장 위치는 앱의 Documents 폴더이며, Finder/iTunes 파일 공유를 통해 사용자가 직접 파일을 넣거나 꺼낼 수 있습니다.
  • Documents 폴더는 자동 iCloud/기기 백업 제외 처리가 적용됩니다.
  • 다만 데이터베이스 마이그레이션 또는 가져오기 안전장치로 앱 샌드박스 내부에 별도 SQLite 백업 사본(Application Support/RoutePic/RoutePic.backup.sqlite, Documents/RoutePic.import-backup.sqlite)이 생성될 수 있습니다.
  • 삭제 비밀번호를 설정한 경우 해당 값은 평문이 아닌 해시값 형태로 기기 내에 저장됩니다.
  • 앱은 여행 데이터를 자체 서버나 개발자 서버로 자동 업로드하지 않으며, 클라우드 동기화를 제공하지 않습니다.

3. 외부 서비스와 통신이 발생하는 경우

  • 지도 제공자 — Apple Maps, Google Maps, 또는 MapLibre/MapTiler 온라인 스타일을 사용할 때 지도 렌더링 요청이 해당 제공자 서버로 전달될 수 있습니다.
  • 음성 인식 — 앱은 Apple Speech의 온디바이스 처리를 우선 요청하지만, 사용 언어나 기기 지원 상태에 따라 Apple 음성 인식 서비스와 통신할 수 있습니다.
  • 텍스트 번역 — 사진 텍스트 추출(OCR)은 기기 내에서만 처리되지만, 번역은 Apple Translation을 사용하며 해당 언어팩이 기기에 설치되어 있지 않은 경우 Apple 서버에서 언어팩을 내려받을 수 있습니다.
  • 오프라인 MapLibre 보조 리소스 — 오프라인 지도는 기기 내 MBTiles 파일을 우선 사용하지만, 지도 표시용 글리프/스프라이트 리소스가 로컬에서 준비되지 않은 경우 원격 글꼴·아이콘 리소스로 폴백할 수 있습니다.
  • 메일 및 공유 — 피드백 메일, 여행 데이터 메일 내보내기, 공유 시트는 사용자가 직접 실행한 경우에만 외부 메일 서비스 또는 사용자가 선택한 외부 앱으로 정보가 전달됩니다.

4. 제3자 제공 및 사용자 지시형 공유

  • RoutePic은 광고, 분석, 마케팅 목적으로 사용자 데이터를 제3자와 공유하지 않습니다.
  • 개발자는 앱을 통해 사용자의 기기에 저장된 데이터를 원격으로 수집하거나 열람하지 않습니다.
  • 공유 시트, 메일 전송, Finder/iTunes 파일 공유는 모두 사용자의 명시적 조작으로만 실행됩니다.
  • 사용자가 직접 전송한 메일과 첨부 파일은 해당 수신자가 수신할 수 있습니다.

5. 데이터 보관 및 삭제

  • 앱 샌드박스에 저장된 데이터는 앱 내 삭제 기능 또는 앱 삭제 시 제거됩니다.
  • 단, 사진 앱에 저장된 사진·영상은 iOS 사진 보관함에 별도로 남을 수 있으며 앱 삭제만으로 제거되지 않습니다.
  • 사용자가 메일, 공유 시트, Finder/iTunes 파일 공유로 외부에 복사한 파일은 복사된 위치에서 별도로 관리해야 합니다.

6. 아동 개인정보 보호

RoutePic은 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

개인정보 처리방침

최종 업데이트: 2026년 7월 5일

TripJournal은 오프라인 우선(Offline-First) 앱입니다.
여행 기록 데이터(경로, 사진, 메모 등)는 사용자의 기기 내 로컬 데이터베이스에만 저장됩니다.
단, 온라인 지도 렌더링(Google Maps, MapTiler) 및 음성 인식(Apple) 기능 사용 시 일부 정보가 외부 서버로 전송될 수 있습니다.

1. 수집하는 정보

TripJournal은 여행 기록을 위해 다음 정보에 접근합니다.

  • 위치 정보 — GPS 트래킹으로 여행 경로를 기록합니다. 앱 실행 중 및 백그라운드에서 수집됩니다.
  • 카메라 — 여행 중 사진 및 영상을 촬영합니다.
  • 마이크 — 음성 메모 및 영상 촬영 시 사용됩니다.
  • 사진 라이브러리 — 촬영한 사진을 저장하거나 기존 사진을 불러옵니다.
  • 음성 인식 — 음성 메모를 텍스트로 변환합니다. Apple의 음성 인식 API를 사용하며, 변환 처리 시 오디오 데이터가 Apple 서버로 전송될 수 있습니다. 오프라인 환경에서는 텍스트 변환 기능을 사용할 수 없습니다.

2. 정보의 저장 방식

  • 모든 여행 데이터(경로, 사진, 메모 등)는 기기 내 로컬 데이터베이스(SQLite)에만 저장됩니다.
  • 개발자 또는 제3자 서버로 자동 업로드되지 않습니다.
  • 클라우드 동기화 기능을 제공하지 않습니다.

3. 외부 서비스 사용

TripJournal은 지도 렌더링과 음성 인식 기능에서 아래 외부 서비스를 사용합니다.

  • Google Maps / MapTiler / MapLibre — 지도 렌더링에 사용됩니다. 온라인 지도 타일 로드 시 해당 서비스의 서버에 요청이 전달될 수 있습니다. 오프라인 지도(MapLibre)는 사용자가 기기 내에 직접 등록한 지도 파일을 사용하며 이 경우 외부 서버 통신이 발생하지 않습니다.
  • Apple 음성 인식 — 음성 메모 텍스트 변환 기능 사용 시 오디오 데이터가 Apple 서버로 전송될 수 있습니다. Apple의 개인정보 처리방침은 apple.com에서 확인하세요.

4. 제3자 정보 공유

TripJournal은 사용자 데이터를 광고, 분석, 마케팅 목적으로 제3자와 공유하지 않습니다.

5. 데이터 보관 및 삭제

  • 데이터는 사용자의 기기에 저장되며, 앱 삭제 시 함께 삭제됩니다.
  • 개발자는 사용자의 기기에 저장된 데이터에 접근할 수 없습니다.

6. 아동 개인정보 보호

TripJournal은 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

7. 개인정보 처리방침 변경

본 방침이 변경될 경우 앱 업데이트 또는 이 페이지를 통해 공지합니다. 중요한 변경 사항이 있을 경우 앱 내 알림을 통해 안내할 수 있습니다.
